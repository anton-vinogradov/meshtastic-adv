meshtastic-adv

Keyboard-first Meshtastic client for the M5Stack Cardputer ADV (Cap LoRa-1262).

Web Serial isn't supported here — open this page in desktop Chrome or Edge. Serial access was blocked by the browser.

Before you click

  • Desktop Chrome or Edge only (Web Serial API).
  • Attach the LoRa antenna to the Cap first — never power the PA without it.
  • Use a data USB-C cable, plugged straight into the computer (no hub).
  • If the device isn't listed: hold G0 / BOOT while connecting, then release.
  • After flashing, set your LoRa region & channel (see the project README).
